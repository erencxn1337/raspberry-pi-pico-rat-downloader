# This script is a Rubber Ducky script for a Raspberry Pi Pico.

Works: Windows 11, 10
-
 Total time : 25 sec
-
How to use it?
-
Just edit the download link. You can change the name of the downloaded file if you want. By default, it downloads the file as windowss.exe.

What does it do?
-
Opens Powershell with Administrator permissions (UAC BYPASS)

Adds an exclusion to Defender for the %appdata% folder.

Downloads a payload to Appdata.

Adds a Defender exclusion to the payload.

Runs the payload.

Counts for 4 seconds and closes PowerShell.
